# Publish this profile

This directory is the content for the special GitHub profile repository.

1. Create a **public** repository named exactly `nanbloom001` under the `nanbloom001` account.
2. Copy these files into that repository root.
3. Commit and push to the default branch.

GitHub recognizes the repository because its name exactly matches the account name, then renders `README.md` at the top of the profile overview.
